# KotlinX - try kotlin online

Learn kotlin easy!

### Installing

0. You shoud have installed node js with npm.
1. Clone repo
2. Install node modules from project dir.
3. Start development server.

```
git clone @repo_url
cd project
npm install
npm start
```

For run build - do command: 

```
npm run build
```
